Jsp引入css

Apr 15th, 2013

Servlet和Jsp的NIIT课程已经结束，开始动手自己的实践项目了。

本次项目主题来自 @林桑 的启发，取名为： ‘恋爱ing’，致力于服务广大热恋中的男女朋友。

在这个项目中，有很多知识点是自己没有接触过的，或者是被自己遗忘的，所以也希望能够通过blog来记录自己的开发过程及知识点。

1.jsp中引入css文件（最简单的方式）

在 ’head‘ 标签中加入如下代码：

<link href="mystyle.css" rel="stylesheet" type="text/css"/>

这里的mystyle.css与jsp的文件在同级目录下，当然也可以将css文件放在自定义的文件夹下，不过 href 的路径要写对。

Rails 中文问题

Apr 13th, 2013

虽然是很老的东东，但是还是要记载下来。

在Rails应用中，有时会在代码片段中使用中文，这时需要我们在文件的开头添加如下 code:

Install Rmagick on Ubuntu

Apr 13th, 2013

今天在处理关于使用 ‘carrierwave’ 上传图片的时候，设计到图片的大小问题，需要用到 ‘rmagick’ 这个gem,但是直接添加到Gemfile里然后运行 bundle， 发现未能成功。网上寻找之后发现需要安装一些东东依赖。

具体方案：

sudo apt-get install libmagickwand-dev

然后再安装 ‘rmagick’ 就好了.

gem install rmagick

OK,我的问题解决了！希望你也如此……

参考自：http://superuser.com/questions/163818/how-to-install-rmagick-on-ubuntu-10-04

ROR中使用 Bootstrap-sass

Apr 10th, 2013

对于css等样式设计，我就是个彩笔。所以就想通过某种方式来弥补。前段时间在学习RubyOnRails教程的时候开始接触了bootstrap,但是当时挺懵的，现在在来回顾一下。

假设我们已经建好了Rails项目，然后在Rails应用中使用 bootstrap-sass 进行样式的设计.

1.首先，在Gemfile中添加gem:

gem 'sass-rails', '~> 3.2'
gem 'bootstrap-sass', '~> 2.3.1.0'

2.其次，在自己定义的css.scss文件中引入 ‘bootstrap’

@import "bootstrap";

3.最后，在项目的javascripts（application.js文件）中加入如下代码片段：

// Loads all Bootstrap javascripts
//= require bootstrap

4.最最后，查看bootstrap 中文说明文档了解更多内容。

参考自：github bootstrap-sass

Ubuntu 终端颜色设置

PS1='\[\033[01;32m\]\u@\h\[\033[01;34m\] \w\[\033[01;31m\]$(__git_ps1 "(%s)" \u) \[\033[01;36m\]\$\[\033[00m\] '

2.重启终端即可看到效果。
